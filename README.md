# LitLens — AI Research Assistant

## Quick Start

### Windows
1. Double-click **START.bat**
2. It installs dependencies and opens the app at http://localhost:8501

### Mac / Linux
```bash
pip install -r requirements.txt
streamlit run app.py
```

## First Use
1. Get a free API key at https://console.anthropic.com
2. Paste it in the sidebar when the app opens
3. Start adding papers or uploading PDFs!

## Features
- **Add Paper** — paste title + abstract, Claude analyzes it instantly
- **Upload PDF** — drag & drop PDFs, text is extracted and analyzed automatically
- **Library** — view summaries, key findings, implications, methodology, critical view
- **Synthesis** — generate a full literature review, thematic analysis, or research gaps section
- **Discover** — describe your topic, get paper recommendations and search strategies

## Requirements
- Python 3.8+
- Internet connection (for Anthropic API calls)
